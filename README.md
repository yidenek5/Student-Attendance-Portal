Student Attendance System

A web-based attendance management system built with Jakarta EE, JSP, Servlets, JDBC, and MySQL. 
The system helps educational institutions manage student attendance records, track attendance efficiently, and generate reports.

Features
User Authentication
Student Management
Attendance Tracking
PDF & Excel Report Export
MySQL Database Integration
Technologies

Java 17, Jakarta EE 10, JSP, Servlets, JDBC, MySQL, Maven, Apache Tomcat 10, HikariCP, OpenPDF, and Apache POI.
