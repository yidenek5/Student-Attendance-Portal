# Student Attendance System

A web-based attendance management system developed using Jakarta EE, JSP, Servlets, JDBC, MySQL, and Apache Tomcat. The system enables institutions to manage student attendance records efficiently and generate attendance reports.

## Features

* User Authentication
* Student Management
* Attendance Tracking
* PDF and Excel Report Export
* MySQL Database Integration

## Technologies

* Java 17
* Jakarta EE 10
* JSP & Servlets
* MySQL
* Maven
* Apache Tomcat 10
* HikariCP
* OpenPDF
* Apache POI

## Run the Application

1. Build the project:

```bash
mvn clean package
```

2. Deploy `AttendanceSystem.war` from the `target` folder to Tomcat's `webapps` directory.

3. Start Tomcat and open:

```text
http://localhost:8080/AttendanceSystem/
```
